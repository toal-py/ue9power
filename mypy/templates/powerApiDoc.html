API Documentation

Available modes:

mode explanation
d "mode=d". Single daily value. "d" is the default mode.
w "mode=w". Value of the whole week and values of the week days. The date specifies which week is returned.
m "mode=m". Two display modes: full_month: true and full_month: false. Full_month: true shows value of whole month, full_month: false returns all days that have passed in the specified month (incl. their values) and the preliminary value of the month.
